# Blood-RE
Matching decompilation of Blood (1997) by nukeykt.

Watcom 10.6 and TASM 3.1 are required to build.

## Build instructions
1) Build helix32 and qtools (i.e. `cd helix32` and then `wmake`)
2) Build blood (i.e. `cd blood` and then `wmake`)
